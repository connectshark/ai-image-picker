## 總攬
### 專案簡介
這個網站是一個創新的ai製圖平台，讓使用者可以用google帳號快速註冊並開始使用。

只要輸入一些提詞，ai就會根據提詞的內容自動產生適合的圖片，無需任何繪圖技巧或軟體。
